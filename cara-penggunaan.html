Panduan Penggunaan Aplikasi “V.?_bY.zK x GPT”

Halaman ini dibuat untuk membantu pengguna memahami cara operasional aplikasi pribadi zK — aplikasi berbasis Python/Tkinter yang mendukung pencatatan data pegawai, LKH harian–bulanan–tahunan, dan fitur PDF otomatis.

1. Persiapan Awal

• Pastikan Folder “data” Lengkap

Aplikasi membutuhkan folder /data yang berisi:

  • profil.json
  • atasan.json
  • penilai.json
  • pekerjaan.json
  • lkh.json
  • Gambar background (bg_dashboard.png, bg_profil.png, dll)

• Jalankan Aplikasi

Buka file main_final.py menggunakan Python 3.10+. Aplikasi akan menampilkan halaman dashboard dengan data waktu realtime.

2. Mengisi Data Pegawai

• Buka Menu “Profil”

Klik tombol Profil pada sidebar. Masukkan data seperti:

  • Nama Pegawai
  • NIP
  • Jabatan
  • Unit Kerja

Simpan → Data tampil otomatis di Dashboard.

3. Mengelola Uraian Pekerjaan

• Buka Panel Uraian

Di halaman Profil, klik tombol “+ Uraian Kerja”. Masukkan uraian pekerjaan sesuai tupoksi masing-masing.

Semua data langsung tersimpan ke pekerjaan.json.

4. Mengisi LKH Harian

• Buka Menu “LKH”

Masukkan:

  • Tanggal
  • Uraian pekerjaan
  • Jumlah (Jml)

Simpan → otomatis masuk database lkh.json.

5. Export PDF (Harian, Bulanan, Tahunan)

• Klik Tombol “PDF”

Sistem akan membuat file PDF lengkap dengan:

  • Header logo KUA
  • Tabel rapi
  • Align teks otomatis

PDF tersimpan di folder yang sama.

6. Penyimpanan EXE

• Jika dijadikan EXE

Pastikan folder /data ikut diletakkan di samping file EXE. Jika tidak, background + JSON tidak terbaca.

Halaman ini akan terus diperbarui sesuai perkembangan fitur aplikasi.
